# MeTTa TextMate Grammar

TextMate grammar for the [MeTTa](https://github.com/trueagi-io/hyperon-experimental) programming language, packaged for use with [GitHub Linguist](https://github.com/github-linguist/linguist).

## About MeTTa

**MeTTa** (Meta Type Talk) is a programming language developed by [SingularityNET](https://singularitynet.io/) / [OpenCog Hyperon](https://github.com/trueagi-io/hyperon-experimental) for cognitive computing and artificial general intelligence research.

## Scope

- **Scope name:** `source.metta`
- **File extension:** `.metta`

## Features

Highlights the following MeTTa constructs:

- Comments (`;`)
- Strings (`"..."`)
- Numbers (integers and floats)
- Variables (`$x`, `$head`)
- Operators: `=` (definition), `:` (type annotation), `->` (arrow type), `!` (evaluation)
- Core forms: `match`, `let`, `let*`, `if`, `case`, `import!`, `include`, `bind!`, `pragma!`
- Built-in functions: `get-type`, `collapse`, `superpose`, `assertEqual`, `car-atom`, `cdr-atom`, etc.
- Built-in types: `Type`, `Atom`, `Symbol`, `Expression`, `Number`, `Bool`, `String`
- Booleans: `True`, `False`
- Space references: `&self`, `&space`
